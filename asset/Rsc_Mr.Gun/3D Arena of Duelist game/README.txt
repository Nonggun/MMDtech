
HOW TO PLAY:
•  Control the player by pressing WASD.

•  Jump by passing Space bar.

•  Run by passing Left shift.

•  Crouch by passing Left Ctrl.

•  Attack the enemy by using mouse click.

	  o  Using right click to aim.
	  o  Using left click to fire the gun and weapon.

•  Walking through the object on the floor in order to pick up weapons and med kit.

•  Choosing and changing the weapon by passing TAP and select the weapons to use by using mouse click on the weapon.

•  Dropping the weapon by passing F.

•  Split- screen by passing C.

•  Pause the game by passing Escape.
